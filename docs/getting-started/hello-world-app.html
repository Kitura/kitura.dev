Step 1: Create project

Open the Kitura macOS app from your application folder.

Hover over the skeleton tile and click create:

Choose the directory you wish for the project to be saved.

Once you're happy with the destination click create.

The macOS app will now create and build your project. It will also generate you an xcode project.

Open the Xcode project
