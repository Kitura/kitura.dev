Step 2: Create a project

There's three types of projects you can create using the Kitura app.

  1. Skeleton - A project containing only Kitura and some example code
  2. Basic - A cloud ready project containing every you need to create a hostable server
  3. API - A cloud ready project containing every you need to create a hostable server which is testable using OpenAPI

Select which type of project you'd like to create by selecting the corresponding tile.

Then you'll be asked to select a destination to save the project.

Once you've chosen your desired destination click the create button.

The project will be created and built for you.
