Hello World

This guide uses the command line via the Terminal.

If you'd prefer a UI based experience you can follow the Hello World example using the Kitura macOS app.

However if you'd prefer a more hands on approach you can follow the Hello World example using SPM.

Step 1: Create project

Create a directory for your project:

mkdir HelloWorld && cd HelloWorld

Create a Kitura application using the Kitura CLI:

kitura basic

`kitura basic` will create a Kitura application in your current directory and also builds your application.

Step 2: Run your project

`kitura basic` adds a Hello World example for you in `Sources/HelloWorld/main.swift`.

To run the project:

swift run

Now if you go to:

localhost:8080

You should see `Hello, World!` in the browser.
