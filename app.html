Kitura - macOS App

Get started with Kitura quickly using our new native macOS app.

There's no need to use the command line as the whole process, from package resolution to Xcode project generation, is automated.

RECENT CHANGES

  • v1.1.1: We've made a few UI changes, added a great new app icon, added Notification Center notifications for when your project finishes being built, and Touch Bar support for Macs that support it.
  • v1.1: We've added a third tile which allows you to create Kitura projects with built-in OpenAPI support.
  • v1.0.1: When you close the window, the app quits plus various bug fixes.
  • v1.0.0: Initial Release.
